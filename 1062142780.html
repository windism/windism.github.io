(Leetcode 2021 刷题计划) 82. 删除排序链表中的重复元素 II

本文最后更新于：2021年3月26日 下午

每日一题时间: 2020-03-25
题目链接: 82. 删除排序链表中的重复元素 II
官方题解链接: 删除排序链表中的重复元素 II

题目

存在一个按升序排列的链表，给你这个链表的头节点 head ，请你删除链表中所有存在数字重复情况的节点，只保留原始链表中 没有重复出现 的数字。

返回同样按升序排列的结果链表。

示例 1：
输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]

示例 2：
输入：head = [1,1,1,2,3]
输出：[2,3]

解题方法

博主代码

解题思路: 判断当前节点重复频次,超过一次的时候进行剔除

class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode* dummyHead = new ListNode(-101);
        ListNode* preNode = dummyHead;
        int count = 0;
        dummyHead->next = head;
        while (head != nullptr) {
            if(preNode->next->val == head->val) {
                count++;
            } else {
                if (count == 1) {
                    preNode = preNode->next;
                } else {
                    preNode->next = head;
                }
                count = 1;
            }
            head = head->next;
        }
        if (count > 1) 
            preNode->next = nullptr;
        return dummyHead->next;
    }
};
  • 复杂度分析
    • 时间复杂度: O(N)
    • 空间复杂度: O(1)

题解解法及扩展

解题思路: 如果遇到重复值即进行删除, 直至不同值的节点, 该写法比较适合释放节点空间的操作

class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        if (!head) {
            return head;
        }
        
        ListNode* dummy = new ListNode(0, head);

        ListNode* cur = dummy;
        while (cur->next && cur->next->next) {
            if (cur->next->val == cur->next->next->val) {
                int x = cur->next->val;
                while (cur->next && cur->next->val == x) {
                    // 若需要释放节点空间, 则进行该代码
                    ListNode *deleteNode = cur->next;
                    cur->next = cur->next->next;
                    // 若需要释放节点空间, 则进行该代码
                    delete deleteNode;
                }
            }
            else {
                cur = cur->next;
            }
        }

        return dummy->next;
    }
};
  • 复杂度分析
    • 时间复杂度: O(N)
    • 空间复杂度: O(1)
