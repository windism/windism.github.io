(Leetcode 2021 刷题计划) 131. 分割回文串

本文最后更新于：2021年3月23日 中午

每日一题时间: 2020-03-07
题目链接: 131. 分割回文串
官方题解链接: 分割回文串

题目

给定一个字符串 s,将 s 分割成一些子串,使每个子串都是回文串.

返回 s 所有可能的分割方案.

示例:
输入: "aab"
输出:
[
  ["aa","b"],
  ["a","a","b"]
]

解题方法

这道题思路很容易想到,但是我实际上实施时还是比较胆怯这种回溯题.

回溯法

回溯法: 一种通过探索所有可能的候选解来找出所有的解的算法.如果候选解被确认不是一个解（或者至少不是最后一个解）,回溯算法会通过在上一步进行一些变化抛弃该解,即回溯并且再次尝试.

如果对回溯法了解和使用不是很好,可以练习经典题型: Leetcode 46. 全排列

class Solution {
private:
    vector<vector<string>> res;
    vector<string> cur;

    bool isPalindrome(string &s, int left, int right) {
        while (left < right) {
            if(s[left] != s[right]) {
                return false;
            }
            ++left;
            --right;
        }
        return true;
    }

    void dfs(string &s, int start) {
        if (start >= s.size()) {
            res.push_back(cur);
            return;
        }
        for (int end = start; end < s.size(); ++end) {
            if (isPalindrome(s, start, end)) {
                cur.push_back(s.substr(start, end - start + 1));
                dfs(s, end + 1);
                cur.pop_back();
            }
        }
    }
public:
    vector<vector<string>> partition(string s) {
        dfs(s, 0);
        return res;
    }
};
  • 复杂度分析
    • 时间复杂度: O(N 2^N)
      • 2^N: 为由于字符串长度导致的判断与否的可能性
      • N: 遍历该回文串
    • 空间复杂度: O(N)
      • 注: 惯例不考虑返回结果占用空间

回溯法+记忆化

优化: 减少对于回文串的判断, start 到 end 是否是回文串可以由 start + 1 和 end - 1 是否是回文串状态转移可知

class Solution {
private:
    vector<vector<int>> dp;
    vector<vector<string>> res;
    vector<string> cur;

    int isPalindrome(string &s, int left, int right) {
        if (left >= right) dp[left][right] = 1;
        if (dp[left][right] == 0) {
            dp[left][right] = s[left] != s[right] ? -1 : isPalindrome(s, left + 1, right - 1);
        }
        return dp[left][right];
    }

    void dfs(string &s, int start) {
        if (start >= s.size()) {
            res.push_back(cur);
            return;
        }
        for (int end = start; end < s.size(); ++end) {
            if (isPalindrome(s, start, end) == 1) {
                cur.push_back(s.substr(start, end - start + 1));
                dfs(s, end + 1);
                cur.pop_back();
            }
        }
    }
public:
    vector<vector<string>> partition(string s) {
        dp.assign(s.size(), vector<int>(s.size()));
        dfs(s, 0);
        return res;
    }
};
  • 复杂度分析
    • 时间复杂度: O(N 2^N)
      • 2^N: 为由于字符串长度导致的判断与否的可能性
      • N: 遍历该回文串
    • 空间复杂度: O(N)
      • 注: 惯例不考虑返回结果占用空间
