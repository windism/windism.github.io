(Leetcode 2021 刷题计划) 331. 验证二叉树的前序序列化

本文最后更新于：2021年3月23日 中午

每日一题时间: 2020-03-12
题目链接: 331. 验证二叉树的前序序列化
官方题解链接: 验证二叉树的前序序列化

题目

序列化二叉树的一种方法是使用前序遍历.当我们遇到一个非空节点时,我们可以记录下这个节点的值.如果它是一个空节点,我们可以使用一个标记值记录,例如 #.

     _9_
    /   \
   3     2
  / \   / \
 4   1  #  6
/ \ / \   / \
# # # #   # #

例如,上面的二叉树可以被序列化为字符串 "9,3,4,#,#,1,#,#,2,#,6,#,#",其中 # 代表一个空节点.

给定一串以逗号分隔的序列,验证它是否是正确的二叉树的前序序列化.编写一个在不重构树的条件下的可行算法.

每个以逗号分隔的字符或为一个整数或为一个表示 null 指针的 '#' .

你可以认为输入格式总是有效的,例如它永远不会包含两个连续的逗号,比如 "1,,3".

示例 1:
输入: "9,3,4,#,#,1,#,#,2,#,6,#,#"
输出: true

示例 2:
输入: "1,#"
输出: false

示例 3:
输入: "9,#,#,1"
输出: false

解题方法

当时看到题目的标签是栈, 其实仔细想想就会想到第一种解法, 第二种方法确实学到啦!

栈

前序遍历是按照根节点-左子树-右子树

9,3,[4,#,#],1,#,#,2,#,6,#,#(遇到两个#, 剔除父节点并转为#, 注意从头部)
9,3,#,[1,#,#],2,#,6,#,#(重复)
9,[3,#,#],2,#,6,#,#(重复)
9,#,2,#,[6,#,#](重复)
9,#,[2,#,#](重复)
[9,#,#](重复)

相对巧妙处理方式可以通过计数的方法, 数字代表必定存在两个子节点, # 代表无子节点且废弃父节点的一个子节点名额.

class Solution {
public:
    bool isValidSerialization(string preorder) {
        int i = 0, N = preorder.size();
        stack<int> stk;
        stk.push(1);
        while (i < N) {
            if (stk.empty()) {
                return false;
            }
            if (preorder[i] == ',') {
                i++;
            } else if (preorder[i] == '#'){
                stk.top() -= 1;
                if (stk.top() == 0) {
                    stk.pop();
                }
                i++;
            } else {
                while (i < N && preorder[i] != ',') {
                    i++;
                }
                stk.top() -= 1;
                if (stk.top() == 0) {
                    stk.pop();
                }
                stk.push(2);
            }
        }
        return stk.empty();
    }
};
  • 复杂度分析
    • 时间复杂度: O(N)
    • 空间复杂度: O(N)

计数

计数的方式是通过统计二叉树的节点数量进行计算的.例如对于一个父节点,每新增一个数字子节点就多需要一个空子节点.

class Solution {
public:
    bool isValidSerialization(string preorder) {
        int i = 0, N = preorder.size(), count = 1;
        while (i < N) {
            if (count <= 0) return false;
            if (preorder[i] == ',') {
                ++i;
            } else if (preorder[i] == '#') {
                count -= 1;
                ++i;
            } else {
                while (i < N && preorder[i] != ',') {
                    ++i;
                }
                count += 1;
            }
        }
        return count == 0;
    }
};
  • 复杂度分析
    • 时间复杂度: O(N)
    • 空间复杂度: O(1)
